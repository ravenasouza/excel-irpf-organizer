# Organizador de Documentação para Imposto de Renda em Excel
Este projeto foi desenvolvido como parte do desafio prático "Criando um Organizador de Declaração de Imposto de Renda" no curso Santander - Excel com Inteligência Artificial na DIO. A ferramenta foi desenhada para centralizar todos os dados necessários para a declaração anual, otimizando o fluxo de coleta de informações.

## Estrutura da Ferramenta
A planilha foi dividida logicamente para facilitar a navegação e garantir que nenhum dado essencial seja esquecido:

* **Dados do Titular:** Centralização de informações cadastrais (Nome, CPF, Título de Eleitor, Endereço e Contatos) e status de residência/dependência.
* **Informes de Rendimentos:** Controle detalhado de saldos bancários e anexos de documentos (PDFs), com somatórios automáticos para facilitar o preenchimento no programa da Receita.
* **Notas e Extratos:** Registro cronológico de entradas (Freelance, Holerite, CNPJ) com categorização e valores.

## Funcionalidades Aplicadas
* **Segregação de Dados:** Separação clara entre dados estáticos (cadastro) e dados dinâmicos (movimentações financeiras).
* **Automação de Cálculos:** Uso de funções de soma para consolidar o patrimônio declarado em diferentes instituições financeiras.
* **Gestão de Documentos:** Campo dedicado para controle de anexos, garantindo que o usuário saiba exatamente qual arquivo corresponde a cada lançamento.
* **Interface Limpa:** Design focado na legibilidade, utilizando uma estrutura de tópicos para reduzir o erro humano no preenchimento.

## Objetivos de Aprendizado
* Estruturação de dados complexos em tabelas simples.
* Aplicação de lógica de organização para processos burocráticos.
